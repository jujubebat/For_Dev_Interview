# 운영체제

**운영체제**
- 운영체제는 하드웨어와 소프트웨어를 연결 해주는 소프트웨어다.
- 운영체제의 기능
  - 컴퓨터 시스템 자원을 효율적으로 관리
  - 컴퓨터 시스템을 편리하게 사용할 수 있는 환경 제공
  - 악성 프로그램에 대해 사용자와 운영체제 자신을 보호

**멀티 프로세싱 시스템**
- 다수의 CPU(프로세서)가 협렵적으로 작업들을 동시에 처리하는 것
- 각 프로세서는 다수의 프로세스를 처리한다.


**멀티 프로그래밍 시스템**
- 메모리 공간을 분할해 여러 프로그램들을 동시에 메모리에 올려놓고 처리하는 것
